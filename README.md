## üí° Usage

### Generating an Excuse

1. **Select a category** from the grid (University, Work, Friends, etc.)
2. Click the **Generate Excuse** button
3. Wait for the AI to generate a creative excuse
4. The excuse appears in a card with a **typewriter animation**

### Saving Favorites

- Click the **Save** button on any excuse card to add it to your favorites
- All favorites are listed at the bottom of the page
- Click the **heart** icon again or the trash icon to remove a favorite
- Use **Clear all** to delete all saved favorites at once

### Copying an Excuse

- Click the **Copy** button on any card
- The text is copied to your clipboard
- A toast notification confirms the action

### Changing Language

- Click the **üåê EN | ES** toggle in the top-right corner of the header
- The entire interface switches between English and Spanish
- Excuses generated after the switch will be in the selected language
- Your language preference is saved and persists across page reloads

## üîå API Reference

### `POST /api/generate-excuse`

Generates a funny excuse using the OpenAI API.

**Request body:**

```json
{
  "category": "Work",
  "language": "en"
}
```

| Field | Type | Values | Description |
|---|---|---|---|
| `category` | `string` | `University` \| `Work` \| `Friends` \| `Family` \| `Relationship` \| `Absurd` | The excuse category |
| `language` | `string` | `en` \| `es` | Output language for the excuse |

**Success response `200`:**

```json
{
  "excuse": "My dog accidentally submitted my resignation letter while I was napping.",
  "category": "Work"
}
```

**Error response `400` / `500`:**

```json
{
  "error": "Invalid or missing category"
}
```
